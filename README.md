## 📖 معرفی

پروژه **مدیریت گیم‌نت** یک ابزار تحت وب برای مدیریت کامل گیم‌نت‌ها و کافی‌نت‌هاست. با این سیستم می‌توانید:

- جلسات بازی را مدیریت کنید (شروع، بستن، تغییر دسته، انتقال میز)
- میزها را رزرو کنید و از تداخل با جلسات فعال جلوگیری کنید
- دستگاه‌ها و کنسول‌ها را با قیمت‌گذاری ساعتی تعریف کنید
- سفارشات کافه را ثبت و موجودی انبار را مدیریت کنید
- کاربران را با دو سطح دسترسی (ادمین/سوپرادمین) مدیریت کنید
- گزارش‌های روزانه را به صورت Excel دریافت کنید

سیستم با **Node.js** و **Express** نوشته شده، از **MongoDB** استفاده می‌کند و رابط کاربری کاملاً واکنش‌گرا با **HTML/CSS/JS** خالص پیاده‌سازی شده است.

## ✨ امکانات کلیدی

### 🎯 مدیریت جلسات
- شروع و بستن جلسات با محاسبه خودکار هزینه
- تغییر دسته (Mode) و انتقال میز در حین بازی
- ادامه جلسه (Reactivate) پس از بسته شدن
- تایمر معکوس برای کنترل زمان بازی
- ثبت پیش‌پرداخت

### 📅 سیستم رزرو
- رزرو میز با نام و شماره مشتری
- شروع خودکار از رزرو
- هشدار تداخل با جلسات فعال
- اطلاع‌رسانی رزروهای نزدیک  

### 🖥️ مدیریت دستگاه‌ها
- تعریف دستگاه‌ها با کنسول‌های مختلف (PS4, PS5, Xbox, PC, ...)
- قیمت‌گذاری ساعتی (دسته‌ای، ثابت، نفر-ساعتی)
- وضعیت‌های ویژه: VIP 👑, Royal 💎, Legendary 🌟

### ☕ مدیریت کافه
- تعریف آیتم‌های کافه با قیمت و موجودی
- ثبت سفارش در حین جلسه
- کاهش خودکار موجودی انبار


### 🛡️ امنیت
- احراز هویت JWT با Access/Refresh Token
- رمزنگاری رمز عبور با bcrypt
- چک انقضای کاربر و گیم‌نت در هر درخواست
- رمز داخلی برای بخش‌های حساس

### 🎨 رابط کاربری
- ۱۴ تم رنگی مختلف (تاریک، روشن، سایبرپانک، طلایی، و ...)
- طراحی واکنش‌گرا (Responsive)
- مودال‌های شیشه‌ای (Glassmorphism)
- سلکتورهای جستجوگر و تقویم شمسی

## 📦 نصب و راه‌اندازی

### ۱. کلون کردن مخزن

```bash
git clone https://github.com/your-username/gamenet-manager.git
cd gamenet-manager
```

### ۲. نصب وابستگی‌ها

```bash
npm install
```

### ۳. تنظیم متغیرهای محیطی

```bash
cp .env.example .env
# فایل .env را با اطلاعات خود ویرایش کنید
```

### ۴. اجرا در محیط توسعه

```bash
npm run dev
```

### ۵. ایجاد کاربر سوپرادمین

```bash
node seed.js
```

## 📊 APIهای اصلی

| متد | مسیر | توضیح |
|-----|------|-------|
| `POST` | `/api/v1/auth/login` | ورود کاربر |
| `POST` | `/api/v1/auth/refresh` | تمدید توکن |
| `GET` | `/api/v1/auth/me` | اطلاعات کاربر جاری |
| `GET` | `/api/v1/users` | لیست کاربران (فقط سوپرادمین) |
| `POST` | `/api/v1/users/with-gamnet` | ایجاد کاربر + گیم‌نت |
| `GET` | `/api/v1/devices` | لیست دستگاه‌ها |
| `GET` | `/api/v1/cafe` | لیست آیتم‌های کافه |
| `GET` | `/api/v1/sessions` | لیست جلسات یک روز |
| `POST` | `/api/v1/sessions/start` | شروع جلسه جدید |
| `POST` | `/api/v1/sessions/reserve` | رزرو میز |
| `POST` | `/api/v1/sessions/:id/close` | بستن جلسه |
| `PUT` | `/api/v1/sessions/:id/mode` | تغییر دسته |
| `PUT` | `/api/v1/sessions/:id/table` | انتقال میز |
| `POST` | `/api/v1/sessions/:id/cafe` | سفارش کافه |
| `GET` | `/api/v1/reports/export` | خروجی اکسل روزانه |
